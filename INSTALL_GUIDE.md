# 牵牛花打印助手 - 快速安装指南

## 方法一：直接安装APK（最简单）

### 步骤1：获取APK文件
1. 从开发者处获取 `ChianiuPrinter.apk` 文件
2. 将APK文件传输到Android设备（通过USB、微信、QQ等方式）

### 步骤2：安装APK
1. 在Android设备上找到APK文件
2. 点击APK文件开始安装
3. 如果提示"禁止安装来自未知来源的应用"，请：
   - 点击"设置"
   - 开启"允许来自此来源的应用"
   - 返回继续安装

### 步骤3：初始设置
1. 打开"牵牛花打印助手"应用
2. 按照提示完成以下设置：
   - 开启无障碍服务
   - 连接蓝牙打印机
   - 授予必要权限

## 方法二：通过Android Studio安装（开发者）

### 环境要求
- Android Studio 2022.3.1 或更高版本
- Android SDK 34
- Java 17

### 安装步骤
1. 打开Android Studio
2. 选择"Open" -> 选择项目文件夹
3. 等待Gradle同步完成
4. 连接Android设备（确保开启USB调试）
5. 点击运行按钮（▶️）
6. 选择目标设备
7. 等待构建和安装完成

## 方法三：使用ADB命令行安装

### 环境要求
- ADB工具（Android SDK Platform-Tools）
- 已开启USB调试的Android设备

### 安装步骤
```bash
# 1. 连接设备
adb devices

# 2. 安装APK
adb install ChianiuPrinter.apk

# 3. 重新安装（如果已存在）
adb install -r ChianiuPrinter.apk

# 4. 启动应用
adb shell am start -n com.example.chianiuprinter/.MainActivity
```

## 权限配置说明

### 1. 无障碍服务（必须）
**作用**：监听牵牛花App界面操作

**开启方法**：
1. 打开Android系统设置
2. 进入"无障碍"或"辅助功能"
3. 找到"牵牛花打印助手"
4. 开启开关

### 2. 蓝牙权限（必须）
**作用**：连接德佟P1打印机

**开启方法**：
1. 应用首次连接打印机时会自动请求
2. 点击"允许"
3. 对于Android 12+，还需要位置权限

### 3. 位置权限（Android 6.0+）
**作用**：搜索蓝牙设备

**开启方法**：
1. 应用首次搜索打印机时会请求
2. 点击"允许"
3. 可以选择"仅在使用时允许"

### 4. 前台服务权限（必须）
**作用**：在后台持续运行

**开启方法**：
1. Android 9.0+会自动请求
2. 点击"允许"

### 5. 开机自启权限（可选）
**作用**：开机自动启动服务

**开启方法**：
1. 不同设备设置位置不同
2. 通常在"应用管理" -> "自启动管理"
3. 找到"牵牛花打印助手"并开启

## 打印机连接指南

### 1. 打印机准备
1. 确保德佟P1打印机已充电
2. 装入58mm热敏纸
3. 开启打印机电源

### 2. 蓝牙配对
1. 进入Android系统设置 -> 蓝牙
2. 开启蓝牙
3. 搜索设备
4. 找到"德佟P1"或类似名称的设备
5. 点击配对（可能需要输入密码，通常是0000或1234）

### 3. 应用内连接
1. 打开"牵牛花打印助手"
2. 点击"连接打印机"
3. 选择已配对的德佟P1打印机
4. 等待连接成功

## 测试打印

## 常见问题解决

### Q1：安装时提示"解析包错误"
A：可能是APK文件损坏或设备不兼容，请重新下载或联系开发者。

### Q2：无法开启无障碍服务
A：检查设备是否已开启"开发者选项"，部分设备需要先开启开发者选项。

### Q3：搜索不到打印机
A：
1. 检查打印机是否已开机
2. 检查打印机是否处于配对模式（通常需要长按某个按钮）
3. 重启手机蓝牙
4. 靠近打印机（蓝牙有效距离约10米）

### Q4：打印乱码
A：
1. 检查打印机字符编码设置
2. 确保打印机支持中文
3. 尝试重启打印机

### Q5：应用在后台被关闭
A：
1. 关闭电池优化
2. 锁定应用（多任务界面长按应用图标）
3. 开启自启动权限
